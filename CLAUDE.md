# Trulečník

Bilingual (Czech default at `/`, English at `/en/`) static Astro site
documenting the rules of **Trulečník** — a three-player cut-throat billiards
variant. Deployed to **trulecnik.cz** via Cloudflare Pages. The full ruleset
lives in `src/data/rules.ts` (both languages side by side); UI chrome strings in
`src/i18n/ui.ts`.

## Git workflow

- Commit directly to `main`. Do not create a feature branch first.
